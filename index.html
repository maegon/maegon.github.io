- 차트목록 (2/2) : 장르별 차트 음악 장르에 따라 구분하여 음악을 리스팅하였습니다.
음악 장르에 대한 기준은 사용자가 음악을 업로드할 때 음악 장르를 선택하게 되어 있습니다.
선택된 장르에 따라 리스팅된다고 보시면 됩니다.
시연 영상
공통 트러블 슈팅 : 차트목록에서의 담기 기능
문제 : TOP100.html이나 genreChart.html에서 '담기' 클릭 시 회원 개인의 플레이리스트에 음악이 추가되는 기능을 컨트롤러에서 구현은 했지만 실제 프론트단에서는 해당 음악 목록이 나타나지 않음
원인 : 회원 당 하나의 플레이리스트를 가져오는데 컨트롤러에서 플레이리스트를 가져올 때 List<>으로 가져옴
해결 : 실제로 음악 담기 기능이 정확히 동작하는 지 파악하기 위해 @OneToMany 어노테이션이 적용된 해당 컬럼은 @ManyToMany로 변경하여 DB에서 리스트를 볼 수 있도록 하였고, 실제로 담기 기능은 동작하는 것을 확인했음
컨트롤러에서 플레이리스트를 가져올 때 List<>를 PlayList 타입으로 변경해주고 타림리프 문법에 맞게 수정함으로써 해결함
결과 및 소감 :
Apollon 프로젝트를 시작하기 앞서 조원들과 Git-Flow 전략 하나씩 맞춰가며 다양한 오류도 겪어보고
해결도 해보고 나서 프로젝트를 시작하여 깃 브랜치 오류에 대한 대처를 원활히 할 수 있었습니다.

프로젝트의 목표였던 음악을 업로드하고 음악을 웹에서 재생하며 다른 페이지로 이동하였을 때 정지없이 재생되는 것이
목표였습니다. 제 초급 프로젝트때 아쉬웠던 부분을 Apollon을 통해 조원들과 함께 기능들을 가꿔나가며 개발하여
보다 높은 퀄리티로 프로젝트를 완성할 수 있었습니다.

아쉬운 점은 다른 페이지로 로딩하는 짧은 구간동안 음악이 잠깐 끊히고 페이지 로딩이 되면 이어서 재생되었습니다.
하지만 전체적으로 기능이 정상 작동되어 뿌듯하였고 이 프로젝트를 통해 개발자로서의 역량과 관심을
크게 향상시킨 경험이었습니다.

EPOS
EDM 음악 재생 애플리케이션

EPOS는 자바를 배운 것을 토대로 혼자 진행한 첫 프로젝트이자 초급 프로젝트로, 음악 재생 기능을 구현한 애플리케이션입니다.
이 프로젝트는 EDM이라는 음악 장르를 즐기는 제 흥미에서 시작되었으며 자바 프로그래밍 언어를 사용하여 개발하였습니다.
재생, 정지, 반복재생과 같은 음악 컨트롤을 구현하였고 음악 추가와 같은 음악 목록 관리 기능을 제공합니다.
콘솔에서만 구현된 프로그램으로, 명령어 입력을 통해 동작합니다.
개발 기간 : 2024.04.03 ~ 2024.04.23

추천음악을 메인화면에 띄우기, 음악 컨트롤 세분화 등 구현하고 싶은 기능은 많았는데 기술적 한계와 정보 부족으로 인해 구현하지 못했었습니다.
또한, 이제와서 생각해보니 회원가입할 때 너무나도 단조롭게 구현되어 있어 아쉬웠고, 이를 통해 사용자 경험(UX)의 중요성도 배울 수 있었습니다.

그래도 스스로 오직 혼자 개발한 프로젝트라는 의미와 다른 수강생들은 시도하지 않은 개성있는 주제를 선택하여 프로젝트를 진행하고
처음으로 외부 라이브러리를 사용하면서 부족한 부분을 스스로 공부해가며 음악 재생 기능을 구현한 과정은 제게 큰 자신감을 심어주었습니다.

2주 반동안 고민하고 학습하며 작은 문제들을 해결해 나가는 과정에서 개발자로서의 끈기와 문제 해결 능력을 배웠고,
이 모든 경험은 이후 프로젝트에서도 동기가 되었습니다. 비록 초급 프로젝트였지만, 제게는 첫걸을을 내딘 기억에 남는 프로젝트입니다.
